Important FAQ

what is the Document Object Model(DOM)?

DOM represents a webpage as a tree structure, allowing programs to dynamically change content, styles, and structure.

How do you select and element from DOM?

Use methods like getElementById, querySelector, or getElementsByClassName.

What is event delegation in the context of the DOM, and why is it useful?

Attaching a single listener to a parent element to manage child events, improving performance and handling dynamic elements.

How do you manipulate an element's attributes and styles using the DOM?

Use setAttribute, getAttribute, and element.style to modify element attributes and styles.
